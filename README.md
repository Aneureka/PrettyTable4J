# PrettyTable4J

## Feature

- Print your class fields in an elegant way

## How to use

```xml
<dependency>
    <groupId>com.github.aneureka</groupId>
    <artifactId>PrettyTable4J</artifactId>
    <version>0.0.1</version>
</dependency>
```

## For example

```java
@Test
public void print() {
    ObjectTabularPrinter.print("");
}
```
